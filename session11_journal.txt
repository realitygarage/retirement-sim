  - Toggle placement: NW chart header (small button, like "book | liq")
  - Liquidation NW formula per year:
      for each property still held:
        saleNet = marketValue * (1 - sellingCosts)  // 5%
        gain = max(0, saleNet - costBasis)
        married excl = $500K on primary only
        taxableGain = max(0, gain - marriedExcl)
        capGainsTax = taxableGain * (0.238 + 0.044)
        netEquity = saleNet - mortgageBalance - capGainsTax
      liquidationNW = sum(netEquity) + investedCash + sweepSavings - hiDebt
